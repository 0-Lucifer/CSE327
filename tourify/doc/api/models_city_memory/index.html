city_memory

models\city_memory library

City-based memory aggregation models for travel organization.

This library provides the CityMemory class for organizing travel memories by destination. It aggregates photo counts, tracks visit dates, and manages city-specific metadata for the travel diary interface.

Example usage:

final cityMemory = CityMemory(
  cityName: 'Paris',
  photoCount: 25,
  lastVisited: DateTime.now(),
  thumbnailPath: 'assets/paris.jpg',
);
